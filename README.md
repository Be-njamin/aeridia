> Yet another Dark Color Theme for Visual Studio Code.

# Installation

- Open Extensions Tab in Sidebar
  - `⌘ + ⇧ + X`
- Enter the Theme Name in the Search
  - `Aeridia`
- Install and Activate the Theme
  - `⌘ + ⇧ + P`
  - Enter: `Color Theme`
  - Select: `Aeridia`
